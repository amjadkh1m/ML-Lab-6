# Ecommerce Customers - Linear Regression Model

This project applies a Linear Regression approach to estimate customers' yearly spending based on their activity and engagement data.

## Dataset Overview

The dataset includes the following features:

* Average Session Length
* Time Spent on Mobile App
* Time Spent on Website
* Membership Duration

Target Variable:

* Yearly Amount Spent

## Technologies Used

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Scikit-learn
* Jupyter Notebook

## Project Steps

* Import and inspect the dataset
* Perform basic data preprocessing
* Define input features and target variable
* Split the data into training and testing sets
* Train a Linear Regression model
* Generate predictions
* Evaluate the model using MAE, MSE, RMSE, and R² metrics
* Visualize prediction results

## Results and Insights

* Membership duration shows the highest influence on customer spending
* Mobile app usage contributes more significantly than website usage
* The model achieves good predictive performance overall
